## 使用方法

1. 下载本仓库中的资源文件。
2. 打开谷歌浏览器，点击右上角的菜单按钮（三个竖点）。
3. 选择“更多工具” -> “扩展程序”。
4. 在扩展程序页面中，打开右上角的“开发者模式”。
5. 点击“加载已解压的扩展程序”按钮。
6. 选择您下载的资源文件所在的文件夹，点击“选择文件夹”。
7. 插件将自动添加到您的浏览器中，您可以在扩展程序列表中看到它。
